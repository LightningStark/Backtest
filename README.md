# Backtest 
Stock market data backtesting to develop a Trading Strategy, 
Historical data of stocks was downloaded from NSE(india) website

To execute the code successfully
1. python version >3.0 required
2. required python packages: pandas,csv
3. remove the csv files from stocks folder and store them with python project files

output window
1. T - target hit(profit)
2. SL - stoploss hit(loss)
3. Days column defines number of days after which exit from BUY/SELL position was made
4. At the bottom it displays Win ratio, which is percentage of winning trades from total trades
